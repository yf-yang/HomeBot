## 功能概述

游戏手柄控制应用允许你使用 Xbox 手柄同时控制 HomeBot 的底盘和机械臂：

- **同时控制**：左手控制底盘运动，右手控制机械臂，无需模式切换
- **精确操控**：模拟摇杆提供平滑的速度控制
- **一键功能**：手腕水平、夹爪控制等功能一键完成
- **震动反馈**：紧急停止时手柄震动提示
- **安全保护**：Back 键紧急停止，Start 键复位

## 系统要求

- **操作系统**：Windows 10/11（使用 XInput API）
- **Python**：3.11+
- **硬件**：Xbox 手柄（360/One/Series X|S）
- **服务**：底盘服务和机械臂服务必须已启动

## 硬件准备

### 支持的手柄

| 手柄型号 | 连接方式 | 说明 |
|---------|---------|------|
| Xbox 360 手柄 | USB 有线 / 无线适配器 | 经典款，兼容性最好 |
| Xbox One 手柄 | USB 有线 / 蓝牙 | 需 Windows 10+ |
| Xbox Series X\|S 手柄 | USB 有线 / 蓝牙 | 最新款，支持更好 |

### 连接步骤

1. **有线连接**：直接使用 USB 线连接电脑
2. **无线连接**：
   - 按手柄上的 **配对键**（Xbox 按钮旁边的小按钮）
   - 在 Windows 设置 → 蓝牙和其他设备 → 添加设备
   - 选择 "Xbox Wireless Controller"

3. **验证连接**：
   - 按 Xbox 按钮，指示灯常亮表示已连接
   - 在 Windows 游戏控制器设置中可查看状态

## 软件配置

### 1. 基础配置

编辑 `software/src/configs/config.py` 中的 `GamepadConfig`：

```python
@dataclass
class GamepadConfig:
    """游戏手柄控制配置"""
    
    # ========== 底盘控制参数 ==========
    max_linear_speed: float = 0.5          # 最大线速度 (m/s)
    max_angular_speed: float = 1.0         # 最大角速度 (rad/s)
    trigger_deadzone: float = 0.1          # 扳机键死区
    left_stick_deadzone: float = 0.15      # 左摇杆死区
    
    # ========== 机械臂控制参数 ==========
    arm_base_step: float = 3.0             # 基座关节步进 (度/帧)
    arm_elbow_step: float = 2.0            # 肘关节步进 (度/帧)
    arm_shoulder_step: float = 2.0         # 肩关节步进 (度/帧)
    arm_wrist_flex_step: float = 3.0       # 腕屈伸步进 (度/次)
    arm_wrist_roll_step: float = 3.0       # 腕旋转步进 (度/帧)
    arm_gripper_open: float = 90.0         # 夹爪打开角度
    arm_gripper_close: float = 0.0         # 夹爪关闭角度
    arm_speed: int = 800                   # 机械臂运动速度
    right_stick_deadzone: float = 0.15     # 右摇杆死区
    
    # ========== 通信配置 ==========
    chassis_service_addr: str = "tcp://localhost:5556"
    arm_service_addr: str = "tcp://localhost:5557"
    
    # ========== 轮询配置 ==========
    polling_interval: float = 0.02         # 50Hz (20ms)
```

### 2. 参数调优建议

**摇杆死区调整**：
- 如果发现不碰摇杆底盘也会动，增大死区值
- 推荐范围：0.10 ~ 0.25

```python
left_stick_deadzone: float = 0.20   # 增大死区减少漂移
right_stick_deadzone: float = 0.20
```

**速度限制**：
- 新手建议先降低速度，熟悉后再提高

```python
max_linear_speed: float = 0.3       # 降低最大线速度
max_angular_speed: float = 0.6      # 降低最大角速度
```

**机械臂步进**：
- 觉得机械臂移动太快或太慢，调整步进值

```python
arm_shoulder_step: float = 1.0      # 减小步进，更精细控制
arm_elbow_step: float = 1.0
```

## 控制映射

### 底盘控制（左手区）

| 输入 | 功能 | 速度范围 | 说明 |
|-----|------|---------|------|
| 左摇杆 ↑ | 前进 | 0 ~ +0.5 m/s | 推得越猛越快 |
| 左摇杆 ↓ | 后退 | 0 ~ -0.5 m/s | |
| 左摇杆 ← | 左转 | 0 ~ +1.0 rad/s | 原地旋转 |
| 左摇杆 → | 右转 | 0 ~ -1.0 rad/s | |
| LT（左扳机） | 向左平移 | 0 ~ -0.5 m/s | 全向轮特性 |
| RT（右扳机） | 向右平移 | 0 ~ +0.5 m/s | |

> **技巧**：可以同时操作摇杆和扳机，例如左摇杆前进 + RT 右平移 = 斜向移动

### 机械臂控制（右手区）

| 输入 | 功能 | 控制关节 | 效果 |
|-----|------|---------|------|
| 右摇杆 ← | 基座左转 | base | 机械臂整体左转 |
| 右摇杆 → | 基座右转 | base | 机械臂整体右转 |
| 右摇杆 ↑ | 前伸 | elbow | 手臂向前伸出 |
| 右摇杆 ↓ | 后缩 | elbow | 手臂向后收回 |
| 十字键 ↑ | 上升 | shoulder | 手臂整体抬高 |
| 十字键 ↓ | 下降 | shoulder | 手臂整体降低 |
| 十字键 ← | 手腕左转 | wrist_roll | 手腕水平旋转 |
| 十字键 → | 手腕右转 | wrist_roll | 手腕水平旋转 |
| **Y 键** | 手腕上翻 | wrist_flex | 手腕向上弯曲 |
| **A 键** | 手腕下翻 | wrist_flex | 手腕向下弯曲 |
| **B 键** | 手腕水平 | wrist_flex | 自动计算保持末端水平 |
| **RB 键** | 夹爪打开 | gripper | 夹爪张开到 90° |
| **LB 键** | 夹爪关闭 | gripper | 夹爪闭合到 0° |

### 系统控制

| 输入 | 功能 | 说明 |
|-----|------|------|
| **Back 键** | 紧急停止 | 立即停止所有运动，锁定控制 |
| **Start 键** | 系统复位 | 解除急停，底盘停止，机械臂归位 |

## 高级功能

### 1. 手腕一键水平

按下 **B 键**，手腕角度自动计算为：

```
wrist_flex = 180° - shoulder - elbow
```

**原理说明**：
- shoulder = 30°, elbow = 45° → wrist_flex = 105°
- 这样无论手臂处于什么姿态，末端都能保持水平

**使用场景**：
- 搬运物体时保持平稳
- 放置物品时自动调平
- 拍照/扫描时保持传感器水平

### 2. 速度叠加技巧

由于采用同时控制，你可以：

- **斜向移动**：左摇杆前推 + RT 右扳机 = 向右前方移动
- **旋转中移动**：左摇杆前推 + 左/右摇杆左右 = 边走边转
- **协同操作**：底盘移动的同时调整机械臂姿态

### 3. 精细控制模式

需要精细操作时，可以：
- 轻推摇杆（小幅度）获得低速
- 快速点击方向键代替长按（步进控制）
- 调整配置中的步进值减小移动幅度

## 故障排除

### 手柄无法连接

**症状**：启动时提示 "未检测到手柄"

**排查步骤**：

1. **检查手柄电源**
   ```
   按 Xbox 按钮，确认指示灯亮起
   ```

2. **检查 Windows 连接**
   - 设置 → 蓝牙和其他设备
   - 确认 "Xbox Wireless Controller" 显示为已连接

3. **重新配对**
   - 长按 Xbox 按钮 6 秒关闭手柄
   - 再次按 Xbox 按钮开机
   - 如使用蓝牙，删除设备后重新配对

4. **检查驱动**
   - 设备管理器 → Xbox  peripherals
   - 确认无黄色感叹号

### 底盘/机械臂无响应

**症状**：手柄操作正常，但机器人不动

**排查步骤**：

1. **检查服务是否启动**
   ```bash
   # 查看端口监听状态
   netstat -an | findstr 5556
   netstat -an | findstr 5557
   ```

2. **检查配置地址**
   ```python
   # 确认配置中的地址正确
   chassis_service_addr: str = "tcp://localhost:5556"
   arm_service_addr: str = "tcp://localhost:5557"
   ```

3. **查看服务日志**
   - 检查底盘/机械臂服务终端是否有错误输出
   - 确认串口连接正常

### 摇杆漂移

**症状**：不碰摇杆，底盘自己移动

**解决方案**：

增大死区值：

```python
# software/src/configs/config.py
left_stick_deadzone: float = 0.20   # 从 0.15 增大到 0.20
right_stick_deadzone: float = 0.20
```

### 按键无响应

**症状**：某些按键按下无反应

**可能原因**：
- 处于紧急停止状态（按 Start 复位）
- 优先级被占用（查看底盘/机械臂服务日志）
- 手柄电池电量低，更换电池或充电

### 通信超时

**症状**：频繁出现 "通信失败" 警告

**解决方案**：
- 检查 USB 连接是否稳定
- 降低轮询频率：
  ```python
  polling_interval: float = 0.05  # 从 0.02 改为 0.05 (20Hz)
  ```

## 安全注意事项

### ⚠️ 使用前必读

1. **首次使用**
   - 在开阔地带测试，周围无易碎品
   - 先低速熟悉控制，再逐渐提高速度
   - 了解紧急停止按钮位置（Back 键）

2. **运行中**
   - 保持手柄在有效控制距离内（蓝牙约 10 米）
   - 注意机械臂运动范围，避免碰撞
   - 人员远离机器人运动路径

3. **异常情况**
   - 任何异常立即按 **Back 键** 急停
   - 失去连接时会自动停止（1 秒超时）
   - 重新连接后需按 Start 复位

4. **机械臂限位**
   - 注意各关节角度限制
   - 避免连续快速操作导致电机过热
   - 听到异响立即停止检查

### 紧急停止流程

```
1. 按下 Back 键 → 底盘和机械臂立即停止
2. 手柄震动反馈 → 确认急停已触发
3. 检查异常原因 → 排除故障
4. 按 Start 键 → 系统复位，恢复控制
```

# HomeBot Web控制界面介绍

HomeBot Web控制界面是基于网页的远程遥控端，支持通过手机、平板或电脑在局域网内实时操控机器人。

---

## 顶部状态栏

| 状态指示 | 说明 |
|---------|------|
| **WebSocket** | 🟢 绿色 = 与服务器连接正常，🔴 红色 = 未连接 |
| **仲裁器** | 🟢 绿色 = 底盘服务正常，🟡 黄色 = 底盘服务异常 |
| **人体跟随** | 🟢 绿色 = 跟随模式已启动，🔴 红色 = 未启动 |
| **FPS** | 显示视频流帧率 |

---

## 视频显示区域

- **实时视频流**：显示机器人摄像头画面（MJPEG格式）
- **状态标识**：
  - 🟢 **● LIVE** = 视频流正常
  - 🔴 **● OFFLINE** = 视频流断开

## 虚拟摇杆

采用 [nippleJS](https://yoannmoi.net/nipplejs/) 虚拟摇杆，支持触摸屏操作。

### 左手摇杆（绿色）- 底盘控制

| 操作 | 功能 |
|-----|------|
| 向上推 | 前进 |
| 向下推 | 后退 |
| 向左推 | 原地左转 |
| 向右推 | 原地右转 |
| 斜向 | 前进/后退 + 转向组合 |

> **安全机制**：摇杆松开1秒后自动发送停止命令，防止失控。

### 右手摇杆（蓝色）- 机械臂控制

| 操作 | 功能 |
|-----|------|
| 左右推动 | 控制机械臂底座旋转 |
| 上下推动 | 控制机械臂升降 |

> **注意**：人体跟随模式启动时，右手摇杆控制被禁用。

---

## 功能按钮

| 按钮 | 功能说明 |
|-----|---------|
| **紧急停止** | 🔴 立即停止底盘所有运动，并锁定控制。锁定后摇杆失效，需点击"归位"解锁。 |
| **归位** | 🔵 正常状态下：机械臂归位；锁定状态下：解锁紧急停止，恢复控制。 |
| **夹爪** | 🟠 切换夹爪开合状态：半开 ↔ 闭合 |
| **打开跟随** | 🟢 启动/停止人体跟随模式。启动后按钮变红显示"关闭跟随"。 |

---

## 数据展示

摇杆区域上方显示实时控制数据：

| 数据项 | 说明 |
|-------|------|
| **底盘 X** | 底盘前后线速度（m/s） |
| **旋转 Z** | 底盘旋转角速度（rad/s） |
| **Base** | 机械臂底座角度 |
| **高度 Z** | 机械臂末端高度 |
| **夹爪** | 夹爪当前状态（张开/半开/闭合） |

## 使用步骤

1. **确保服务已启动**：底盘服务、视觉服务、Web服务
2. **连接网络**：手机/电脑与机器人在同一局域网
3. **访问界面**：浏览器输入 `http://<机器人IP>:5000`
4. **检查状态**：确认 WebSocket 和仲裁器显示绿色
5. **开始控制**：
   - 左手摇杆控制底盘移动
   - 右手摇杆控制机械臂
   - 点击按钮执行对应功能

---

## 安全须知

⚠️ **紧急停止锁定机制**：
- 点击"紧急停止"后，底盘立即刹车并锁定
- 锁定状态下，摇杆控制无效
- 必须点击"归位"按钮才能解锁

⚠️ **人体跟随模式**：
- 启动后左手摇杆自动失效（防止人工与自动控制冲突）
- 如需人工接管，先点击"关闭跟随"

## 响应式适配

界面自动适配不同设备：

| 设备 | 布局 |
|-----|------|
| 手机（竖屏） | 视频在上，控制区在下，摇杆左右排列 |
| 平板/电脑（横屏） | 视频在左，控制区在右，摇杆上下排列 |

## 控制方式对比

| 特性 | Web控制界面 | [游戏手柄控制](游戏手柄控制使用指南.md) | [AI助手控制](用Picoclaw小龙虾控制HomeBot.md) |

# 游戏手柄控制 - 快速参考

## 🚀 启动命令

```bash
cd software/src
python -m applications.gamepad_control
```

## 🎮 控制映射速查

### 左手区 - 底盘
```
        ↑ 前进
   ← 左转  右转 →
        ↓ 后退
        
LT ← 左平移    右平移 → RT
```

### 右手区 - 机械臂
```
        十字键 ↑ 上升
十字键 ← 腕左转  腕右转 → 十字键
        十字键 ↓ 下降
        
右摇杆 ← 基座左转    基座右转 → 右摇杆
右摇杆 ↑ 前伸        后缩 ↓ 右摇杆

Y键 - 手腕上翻
A键 - 手腕下翻  
B键 - 手腕水平 (自动计算)

RB - 夹爪打开
LB - 夹爪关闭
```

### 系统
```
Back  - 紧急停止 ⚠️
Start - 复位
```

## ⚙️ 常用配置

```python
# software/src/configs/config.py

@dataclass
class GamepadConfig:
    # 速度限制
    max_linear_speed: float = 0.5      # 线速度 (m/s)
    max_angular_speed: float = 1.0     # 角速度 (rad/s)
    
    # 摇杆死区（漂移时增大）
    left_stick_deadzone: float = 0.15
    right_stick_deadzone: float = 0.15
    
    # 机械臂步进
    arm_base_step: float = 3.0         # 基座
    arm_elbow_step: float = 2.0        # 伸缩
    arm_shoulder_step: float = 2.0     # 升降
```

## 🆘 常见问题

| 问题 | 解决 |
|-----|------|
| 手柄连不上 | 按 Xbox 键开机，检查蓝牙配对 |
| 底盘漂移 | 增大 `left_stick_deadzone` |
| 机器人无响应 | 检查底盘/机械臂服务是否启动 |
| 忘记按键 | 按 Back 急停，按 Start 复位 |

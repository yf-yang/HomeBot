## 配置结构概述

配置采用 **Python dataclass** 分层设计，包含以下模块：

| 配置类 | 说明 | 用途 |
|--------|------|------|
| `CameraConfig` | 摄像头配置 | 图像采集参数 |
| `ArmConfig` | 机械臂配置 | 舵机ID、关节限制、待机位置 |
| `ChassisConfig` | 底盘配置 | 三轮全向底盘参数 |
| `ZMQConfig` | 网络配置 | ZeroMQ 通信地址 |
| `LoggingConfig` | 日志配置 | 日志级别设置 |
| `HumanFollowConfig` | 人体跟随配置 | YOLO检测、跟踪、控制参数 |

## 各配置项详细说明

### 摄像头配置 (CameraConfig)

```python
camera:
  device_id: 0          # 摄像头设备ID (0为默认摄像头)
  width: 1920           # 采集宽度 (像素)
  height: 1080          # 采集高度 (像素)
  fps: 30               # 帧率
```

**修改建议：**
- `device_id`: 外接USB摄像头可尝试 `1`、`2` 等
- `width/height`: 如需更高帧率可降为 `1280x720`
- `fps`: 根据摄像头支持调整，一般 `30` 即可

---

### 机械臂配置 (ArmConfig)

```python
arm:
  # 串口配置
  serial_port: "/dev/tty.usbmodem5AE60527771"  # 与底盘共用串口
  baudrate: 1000000
  
  # 舵机ID映射 (1-6号关节)
  base_id: 1          # J1: 基座旋转
  shoulder_id: 2      # J2: 肩关节
  elbow_id: 3         # J3: 肘关节
  wrist_flex_id: 4    # J4: 腕关节屈伸
  wrist_roll_id: 5    # J5: 腕关节旋转
  gripper_id: 6       # J6: 夹爪
  
  # 关节角度限制 (度) - ⚠️ 人工设置，AI勿动
  joint_limits:
    base: (-180, 180)
    shoulder: (0, 180)
    elbow: (0, 180)
    wrist_flex: (-90, 90)
    wrist_roll: (-180, 180)
    gripper: (0, 90)
  
  # 默认速度/加速度
  default_speed: 1000   # 舵机运动速度
  default_acc: 50       # 舵机加速度
  
  # 休息位置/待机位置 (度) - ⚠️ 人工设置，AI勿动
  rest_position:
    base: -90
    shoulder: 0
    elbow: 150
    wrist_flex: 30
    wrist_roll: 0
    gripper: 45
```

**修改建议：**

| 参数 | 说明 | 建议值 |
|------|------|--------|
| `serial_port` | 根据平台修改，见下方[串口配置参考](#各平台串口配置参考) | - |
| `joint_limits` | **修改前务必确认机械结构限位**，防止损坏 | 根据实际机械臂调整 |
| `rest_position` | 服务启动时自动恢复的位置 | 根据使用场景调整 |
| `default_speed` | 越大动作越快 | 500-3250 |
| `default_acc` | 越大启动/停止越急 | 10-100 |

### 底盘配置 (ChassisConfig)

```python
chassis:
  # 串口配置
  serial_port: "/dev/tty.usbmodem5AE60527771"
  baudrate: 1000000
  
  # 舵机ID映射 (三轮全向底盘)
  left_front_id: 7    # 左前轮
  right_front_id: 9   # 右前轮
  rear_id: 8          # 后轮
  
  # 物理参数
  wheel_radius: 0.08      # 轮子半径 (米)
  chassis_radius: 0.18    # 底盘半径 (米)
  
  # 运动限制
  max_linear_speed: 0.5     # 最大线速度 (m/s)
  max_angular_speed: 1.0    # 最大角速度 (rad/s)
  default_wheel_speed: 3250 # 舵机最大转速
  
  # ZeroMQ服务地址
  service_addr: "tcp://*:5556"
```

**修改建议：**

| 参数 | 说明 | 调整建议 |
|------|------|----------|
| `serial_port` | 串口号 | 根据平台修改 |
| `wheel_radius` | 测量实际轮子半径 | 影响速度计算精度 |
| `chassis_radius` | 测量底盘中心到轮子距离 | 影响旋转精度 |
| `max_linear_speed` | 安全限制 | 室内建议 `0.3-0.5` |
| `max_angular_speed` | 安全限制 | 建议 `0.5-1.0` |

### ZeroMQ 网络配置 (ZMQConfig)

```python
zmq:
  chassis_service_addr: "tcp://*:5556"   # 底盘服务地址
  arm_service_addr: "tcp://*:5557"       # 机械臂服务地址
  vision_pub_addr: "tcp://*:5560"        # 视觉服务发布地址
  speech_service_addr: "tcp://*:5570"    # 语音服务地址
```

**修改建议：**
- `*`: 监听所有网络接口（允许远程连接）
- `127.0.0.1`: 仅本地访问（更安全）
- 如需分布式部署，改为实际IP地址

### 人体跟随配置 (HumanFollowConfig)

```python
human_follow:
  # 模型配置
  model_path: "models/yolo26n.onnx"    # YOLO模型路径
  conf_threshold: 0.5                  # 检测置信度阈值 (0-1)
  
  # 跟踪配置
  max_tracking_age: 30                 # 最大丢失帧数
  min_iou_threshold: 0.3               # IoU匹配阈值
  target_selection: "center"           # 目标选择策略: center/largest/closest
  
  # 推理优化
  inference_size: 320                  # 输入分辨率 320x320
  use_half_precision: False            # FP16半精度（需GPU）
  
  # 跟随控制配置
  target_distance: 1.0                 # 目标距离（米）
  kp_linear: 0.5                       # 线速度P系数
  kp_angular: 1.0                      # 角速度P系数
  max_linear_speed: 0.3                # 最大线速度 (m/s)
  max_angular_speed: 0.8               # 最大角速度 (rad/s)
  dead_zone_x: 150                     # 水平死区（像素）
  dead_zone_area: 0.1                  # 面积死区（相对值）
  
  # 安全配置
  timeout_ms: 1000                     # 通信超时 (ms)
  stop_on_lost: True                   # 丢失目标时停止
  search_on_lost: False                # 丢失时旋转搜索
  lost_patience: 30                    # 丢失容忍帧数
  
  # ZeroMQ配置
  chassis_service_addr: "tcp://localhost:5556"
  vision_sub_addr: "tcp://localhost:5560"
```

**修改建议：**

| 参数 | 说明 | 建议值 |
|------|------|--------|
| `conf_threshold` | 越高误检越少但可能漏检 | 0.4-0.6 |
| `target_selection` | 多目标时的选择策略 | `center`/`largest`/`closest` |
| `inference_size` | 越大精度越高速度越慢 | 320(快)/640(准) |
| `kp_linear/kp_angular` | PID控制P参数 | 根据实际响应调整 |
| `dead_zone_x` | 不响应的小范围偏差 | 100-200 |
| `stop_on_lost` | 安全建议保持 `True` | - |
| `search_on_lost` | 丢失后是否自动搜索 | `False`(安全)/`True`(主动) |

## 如何修改配置

### 方法一：直接修改配置文件

1. 打开 `software/src/configs/config.py`
2. 找到对应配置类，修改默认值
3. 保存后重启相关服务生效

```python
# 示例：修改底盘串口
@dataclass
class ChassisConfig:
    serial_port: str = "COM3"  # Windows 改为 COM3
    # ... 其他配置
```

### 方法二：运行时动态设置

```python
from configs import get_config, set_config, Config

# 获取当前配置并修改
config = get_config()
config.chassis.serial_port = "COM3"

# 或者创建新配置
custom_config = Config()
custom_config.chassis.serial_port = "/dev/ttyUSB0"
set_config(custom_config)
```

## 各平台串口配置参考

### Windows

```python
# 查看设备管理器中的 COM 端口号
serial_port: str = "COM3"  # 或 COM4、COM5 等
```

### Linux (树莓派/Ubuntu)

```python
# USB转串口芯片（CH340/CP2102等）
serial_port: str = "/dev/ttyUSB0"  # 或 ttyUSB1

# 原生USB设备（Arduino/STM32等）
serial_port: str = "/dev/ttyACM0"  # 或 ttyACM1
```

**权限设置：**
```bash
# 添加用户到 dialout 组（需重新登录生效）
sudo usermod -a -G dialout $USER
```

### macOS

```python
# USB设备
serial_port: str = "/dev/tty.usbmodem5AE60527771"
# 或
serial_port: str = "/dev/tty.usbserial-XXXX"
```

### 查找串口命令

```bash
# Windows (PowerShell)
[System.IO.Ports.SerialPort]::getportnames()

# Linux/macOS
ls -la /dev/tty*

# 查找具体设备
dmesg | grep tty
```

## 注意事项

### ⚠️ 安全提醒

1. **机械臂关节限位 (`joint_limits`)**
   - 修改前务必确认机械结构实际限位
   - 错误的限位可能导致舵机堵转损坏
   - 注释中标有 **"人工设置，AI勿动"** 的参数请谨慎修改

2. **底盘速度限制 (`max_linear_speed`, `max_angular_speed`)**
   - 室内使用建议不超过 0.5 m/s
   - 首次测试请使用较低速度

3. **紧急停止机制**
   - 配置修改不影响紧急停止功能
   - 紧急停止后需归位解锁才能恢复控制

### 🔧 调试建议

1. **修改后验证**
   ```bash
   cd software/src
   python -c "from configs import get_config; print(get_config())"
   ```

2. **分步测试**
   - 先测试底盘基础运动
   - 再测试机械臂动作
   - 最后测试应用功能

3. **备份配置**
   - 修改前备份 `config.py`
   - 或使用 Git 管理配置变更

# HomeBot 网页控制端使用指南

## 概述

网页控制端允许通过手机、平板或电脑在局域网内遥操作机器人。

- **左手摇杆**：控制机器人底盘（前进/后退/旋转）
- **右手摇杆**：控制机械臂（开发中）

## 架构

```
手机/平板 → WebSocket → Flask服务器 → ZeroMQ REQ → ChassisService
                                                      ↓
                                              直接调用底盘驱动
                                                      ↓
                                               ST3215舵机
```

## 快速开始

### 1. 启动底盘服务

```bash
cd software

# Windows
start_chassis_service.bat --port COM3

# Linux/Mac
./start_chassis_service.sh --port /dev/ttyUSB0
```

### 2. 启动Web控制端

```bash
# 在新终端中
cd software/src
python -m applications.remote_control
```

### 3. 一键启动（推荐）

```bash
cd software
start_all_merged.bat  # Windows
```

### 4. 手机访问

浏览器访问：`http://<机器人IP>:5000`

## 配置

### 默认地址

- **ChassisService**: `tcp://127.0.0.1:5556`
- **Web服务器**: `http://0.0.0.0:5000`

### 串口设置

```python
ChassisConfig(
    port="COM3",           # Windows
    # port="/dev/ttyUSB0", # Linux
    baudrate=1000000,
)
```

## 使用说明

| 操作 | 说明 |
|------|------|
| 左手摇杆 | 前进/后退/旋转 |
| 右手摇杆 | 机械臂（开发中） |
| 紧急停止 | 立即停止所有运动 |

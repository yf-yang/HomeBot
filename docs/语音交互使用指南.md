# HomeBot 语音交互使用指南

本文档介绍如何使用 HomeBot 的语音交互功能。

## 功能概述

HomeBot 语音交互系统包含以下功能：

1. **语音唤醒** - 通过唤醒词激活机器人
2. **语音识别 (ASR)** - 将语音转换为文字
3. **对话理解** - 通过 LLM 理解用户意图
4. **机器人控制** - 调用 MCP 工具控制底盘/机械臂
5. **语音合成 (TTS)** - 将回复转换为语音播报

## 架构设计

```
┌─────────────────────────────────────────────────────────────┐
│                    WakeupASR Service (PUB)                   │
│  麦克风 → [唤醒检测] → [ASR识别] → PUB {"asr_text": "..."}     │
└────────────────────────┬────────────────────────────────────┘
                         │ PUB/SUB (tcp://*:5571)
┌────────────────────────▼────────────────────────────────────┐
│                Speech Interaction App (SUB)                  │
│                                                              │
│  SUB接收 ──→ DialogueManager ──→ TTS Engine ──→ 扬声器      │
│                    │                                         │
│                    └──→ MCP工具 ──→ ZeroMQ ──→ 底盘服务      │
└─────────────────────────────────────────────────────────────┘
```

## 快速开始

### 1. 安装依赖

```bash
cd software
pip install -r requirements.txt
```

### 2. 下载语音模型

首次使用需要下载语音模型（约 100MB）：

```bash
cd software
python tools/download_speech_models.py
```

或仅下载特定模型：
```bash
python tools/download_speech_models.py --model asr      # 仅ASR
python tools/download_speech_models.py --model wakeup   # 仅唤醒
```

### 3. 检查模型

```bash
python start_speech_service.py --check-models
```

### 4. 启动语音服务

#### 方式一：一键启动（推荐）

```bash
cd software
python start_speech_service.py
```

这将启动两个窗口：
- WakeupASR Service（语音唤醒 + 识别）
- Speech Interaction（对话 + TTS）

#### 方式二：手动启动

终端 1 - 启动 WakeupASR 服务：
```bash
cd software/src
python -m services.speech_service wakeup
```

终端 2 - 启动语音交互应用：
```bash
cd software/src
python -m applications.speech_interaction
```

#### 方式三：完整系统启动

如果还需要底盘和视觉服务：
```bash
cd software
python start_system.py
```

然后在另一个终端启动语音交互：
```bash
python start_speech_service.py
```

## 使用方法

### 基本交互流程

1. **唤醒机器人**
   - 说出唤醒词：`"你好小白"`
   - 听到 `"我在"` 提示音表示唤醒成功

2. **发出指令**
   - 唤醒后直接说话，例如：
     - `"向前走一米"`
     - `"向左转90度"`
     - `"停止"`
     - `"现在几点了"`（闲聊）

3. **等待回复**
   - 机器人会执行动作并通过语音回复

### 支持的语音指令

#### 移动控制
| 指令示例 | 执行动作 |
|---------|---------|
| "向前走一米" | 前进 1 米 |
| "向后退半米" | 后退 0.5 米 |
| "向左转90度" | 左转 90 度 |
| "向右转45度" | 右转 45 度 |
| "停止" | 立即停止 |

#### 状态查询
| 指令示例 | 返回信息 |
|---------|---------|
| "你好吗" | 机器人状态 |
| "现在几点" | 当前时间 |

#### 闲聊
- "你好"
- "你是谁"
- "今天天气怎么样"（需配置天气API）

## 配置说明

### 配置文件

配置文件位于 `software/src/configs/config.py`：

```python
# 语音引擎配置
@dataclass
class SpeechConfig:
    wakeup_keyword: str = "你好小白"      # 唤醒词
    wakeup_sensitivity: float = 0.3      # 唤醒灵敏度
    listen_timeout: float = 1.5          # ASR静默超时（秒）
    mic_index: int = 1                   # 麦克风设备索引

# TTS配置
@dataclass
class TTSConfig:
    appid: str = ""                      # 火山引擎AppID
    access_token: str = ""               # 火山引擎Token
    voice_type: str = "zh_female_vv_uranus_bigtts"  # 音色

# LLM配置
@dataclass
class LLMConfig:
    provider: str = "deepseek"           # 提供商
    api_key: str = ""                    # API密钥
    model: str = "deepseek-chat"         # 模型名称
```

### 环境变量

可以通过环境变量覆盖配置：

```bash
export HOMEBOT_LLM_API_KEY="your-api-key"
export HOMEBOT_TTS_APPID="your-app-id"
export HOMEBOT_TTS_TOKEN="your-token"
```

### 麦克风选择

如果系统有多个麦克风，需要设置正确的索引：

```python
# 查看可用麦克风
python -c "import sounddevice as sd; print(sd.query_devices())"

# 设置麦克风索引
mic_index: int = 0  # 根据实际情况修改
```

## 故障排除

### 模型文件缺失

**现象**: 启动时提示模型文件不存在

**解决**:
```bash
python tools/download_speech_models.py
```

### 无法唤醒

**可能原因**:
1. 麦克风未正确连接
2. 麦克风索引设置错误
3. 环境噪音过大

**排查**:
```bash
# 检查麦克风
python -c "import sounddevice as sd; print(sd.query_devices())"

# 测试录音
python -c "import sounddevice as sd; import numpy as np; 
rec = sd.rec(int(3*16000), samplerate=16000, channels=1); sd.wait(); 
print('录音成功' if len(rec) > 0 else '录音失败')"
```

### TTS 无法播放

**可能原因**:
1. 未配置火山引擎 API
2. 扬声器未连接
3. 音频设备被占用

**解决**:
- 在配置中添加 TTS API 密钥
- 或未配置时使用模拟 TTS（仅打印文字）

### 底盘不响应语音指令

**可能原因**:
1. 底盘服务未启动
2. ZeroMQ 连接失败
3. 优先级被其他控制源占用

**排查**:
```bash
# 检查底盘服务是否运行
python -m services.motion_service.chassis_service

# 检查网络连接
telnet localhost 5556
```

## 高级配置

### 自定义唤醒词

1. 编辑 `models/wakeup/keywords.txt`：
```
你好小白 /1e-20/
自定义唤醒词 /1e-20/
```

2. 修改配置：
```python
wakeup_keyword: str = "自定义唤醒词"
```

### 更换 TTS 音色

在火山引擎控制台选择其他音色，修改配置：
```python
voice_type: str = "zh_male_xxx"  # 男声
```

### 调整灵敏度

```python
# 唤醒灵敏度 (0-1)，值越低越容易触发
wakeup_sensitivity: float = 0.3

# ASR 静默超时（秒）
listen_timeout: float = 1.5
```

## 开发指南

### 添加新的 MCP 工具

编辑 `applications/speech_interaction/mcp_server.py`：

```python
@mcp.tool
def my_custom_action(param: str) -> dict:
    """工具描述（会被LLM使用）"""
    try:
        # 执行动作
        result = do_something(param)
        return {"status": "success", "message": "完成"}
    except Exception as e:
        return {"status": "error", "message": str(e)}
```

同时更新 `MCPClientWrapper._get_tools_schema()` 添加工具定义。

### 修改系统提示词

编辑 `dialogue_manager.py` 中的 `_get_system_prompt()`。
